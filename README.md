# Support Ticket SLA Tracker

## Overview
This project is a Django based support ticket management system that helps users raise issues and allows administrators to track and resolve them efficiently while monitoring SLA deadlines.

## Features
- Ticket creation
- Ticket status tracking
- Admin ticket assignment
- SLA monitoring
- Issue resolution workflow

## Tech Stack
- Python
- Django
- SQLite
- HTML
- CSS

## Future Improvements
- Email notifications
- AI ticket classification
- Analytics dashboard
- Priority prediction

## How to Run the Project

1. Clone the repository:
git clone https://github.com/mariataniya611/support-ticket-sla-tracker.git

2. Navigate into project folder:
cd support-ticket-sla-tracker

3. Install dependencies:
pip install -r requirements.txt

4. Run migrations:
python manage.py migrate

5. Start server:
python manage.py runserver

6. Open browser:
http://127.0.0.1:8000
