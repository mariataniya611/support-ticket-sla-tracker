# Support Ticket SLA Tracker

## Overview
This project is a Django based support ticket management system that helps users raise issues and allows administrators to track and resolve them efficiently while monitoring SLA deadlines.

## Features
- Ticket creation
- Ticket status tracking
- Admin ticket assignment
- SLA monitoring
- Issue resolution workflow

## Tech Stack
- Python
- Django
- SQLite
- HTML
- CSS

## Future Improvements
- Email notifications
- AI ticket classification
- Analytics dashboard
- Priority prediction
